多端极速同步方案手册

1. 同步原理

本应用采用免注册的「同步码（Sync Code）」设计。每台设备独享一个唯一的同步秘钥。若要在其他端（如 iOS 手机端、Windows 电脑端）连接并共享您的计划清单，仅需在其他设备上填入此秘钥即可秒级完成云合并。

2. 同步步骤
  1. 在当前设备复制您的 **专属同步码**；
  2. 在另一台设备的本应用中，点击左侧「设置多端同步」；
  3. 在下方粘贴该同步码，并点击「连接并同步」即可。
3. iOS 端无痛安装 (Safari PWA)
  1. 在 iPhone 自带的 **Safari 浏览器** 中打开本网页链接；
  2. 点击浏览器底部的 **"分享"** 按钮（向上箭头的正方形图标）；
  3. 在弹出的功能菜单中，向下滚动并选择 **"添加到主屏幕"**；
  4. 返回主屏幕，点击生成的"逸程计划"应用图标，即可获得无浏览器地址栏、全屏的原生客户端体验。
4. Windows / 桌面端安装 (PWA)
  1. 使用 **Edge 浏览器** 或 **Chrome** 浏览器打开网页；
  2. 点击浏览器最右侧地址栏的 **"安装应用"**（通常为加号或三个点中的安装选项）；
  3. 应用会一键生成桌面图标，并作为单独视窗的 Windows 桌面应用运行，支持多端数据实时云拉取。
