北洋园pt自查

电影

  • [ ] 我确认不是禁转的种子，可以转种

  • [ ] 我确定种子文件如果是蓝光原盘的话，一定是有中文字幕的版本

很简单，看种子简介里面subtitles有没有chinese就可以了，没有中文字幕的蓝光原盘一定要上传到北洋园pt，不然你差一点就要上传成功了

  • [ ] 我确定蓝光原盘/remux资源如果在同等分辨率下没有版本

比如如果已经有1080p remux资源，你就不要在上传其他站点搬来的1080p remux资源，但是如果没有2160p remux资源你是可以搬来的

  • [ ] 我确认remux作品视频码率不低，1080p不低于4000kb/s，2160p不低于8000kb/s

种子的简介里面是有的，如果不是remux作品要求就没有这么严格啦

  • [ ] 我确定我上传的文件不是同一系列但未确定完结的电影合集，以及以某导演、演员等打包的电影合集（按套装售卖的电影合集除外）

  • [ ] 我确定mediainfo是放在截图的前面，简介后面而不是在海报前面

  • [ ] 我确定站内没有更好的源

北洋园pt

基本步骤是：

  1. 从其他站点下载你想要转的资源
  2. 填写发布页面（用引用id简单一点）
  3. 如果种子通过审核以后，从你发布种子的pt站点下载种子进行本地作种

填写发布页面：

在上面那一行里面，点击候选

点击“阅读候选须知后，点此发布候选”

如果要用引用种子id的话，这样去找：

简介的话一般从其他站点搬运即可，一般不做改动，需要注明转自xxx

全部填写完成后，可以点击预览看一看实际效果进行微调

提交成功以后你会跳转到一个界面，显示：

发布资源后会自动重新下载一个带有[TJUPT]前缀的种子文件，如果没有自动下载请手动下载该资源的种子

进入utorrent，选择添加种子文件

显示这样就是做种成功了

当然，你也可以进入web候选界面查看，如果这里显示有一个上传者就是做种成功了

btschool/52pt

btschool的发种有点水，发种最大的特点是要有一个投票环节（elite user以下需要，elite user以上可以直接发种），我发了几个种子，似乎很难不被通过，发种的步骤大差不差，可惜的一点是自动发种体验不好，因为用户等级不够似乎无法直接跳转到发种界面，只能曲线救国，在其他站点的发布界面复制内容去填写候选环节的一些信息了

新发现的auto_feed PT一键转种脚本这个脚本可以直接给你跳转到btschool候选界面的，这个脚本还是可以的哈哈

52pt也是要通过候选的，但是等你等级升上去了就可以不用候选

1ptba/soulvoice

发种更是水，似乎不需要经过审核环节，个人觉得是因为站点的种子数比较少，鼓励大家积极上传种子的，自然对于资源的品质要求就没有那么高了

mt/ttg

这两个都是大站，估摸着发种不会那么简单

自动发种

pt站点的自动转种推荐使用EasyUpload PT一键转种油猴脚本，它能实现“从源站一键跳转到目标站，并自动填充表单”，极大加速发种流程

由于在探索转种的过程中发现mteam上面这个脚本居然不支持，所以找到了一个支持mteam转种的脚本auto_feed PT一键转种脚本，这个似乎也不错

安装步骤

  1. 安装用户脚本管理器：Chrome/Edge 用 Tampermonkey（篡改猴），Firefox 用 Tampermonkey 或 Violentmonkey
  2. 访问脚本页面EasyUpload PT一键转种油猴脚本：点击“安装此脚本” → 确认安装（这一步可能需要科学上网的）

使用流程

  1. 打开源站的 torrent 详情页，会出现一个这样的小白花在转，点击小白花打开设置面板

  2. 转种站点启用: 定义需要在页面显示的转种站点列表

    点击单个的站点链接是实现单个站点转种，记住，点一次就行了，点太多会出一大堆的

  3. 批量转种启用: 点击页面的一键群转按钮将同时打开选中的站点转发

  4. 站点搜索启用: 定义需要在页面显示的快速搜索列表，搜索似乎是通过imdb编号来实现的，我在想，如果有一些站点没有这些编号的咋办

遇到的问题

  1. 如果发现转过去的图片无法显示怎么办？

    可以通过手动上传图片实现，复制无法显示的图片->https://iili.io/ ->把复制的图片上传->得到链接->链接就是红框那一段->bbcode使用[img]…[img]代码嵌入即可

写在最后

  • 任何情况下都不要完全依赖脚本，脚本不可能做到对每个站点都完美适配，发布前需要自己认真检查是否有填写错误或者不符合站点规则的地方，始终遵守目标站的发种规范
  • 插件目前对音乐、MV、动漫的种子转载不支持(分类可能不会自动填写)，需要自己手动填一下
  • 由于TTG的图片加载策略，需要等页面完全加载完整后再点击转载到其他站，否则种子信息会获取不完整
  • 转的种子不要乱转，最好是free的，如果计算了你的下载量的，你很亏的
