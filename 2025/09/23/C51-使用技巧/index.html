键位

位置 键位 作用
键盘 tab 缩进，使得代码可读性增强
键盘 ctrl+home 跳转到开头
键盘 ctrl+end 跳转到结尾
KEIL configuration 调整字体样式
KEIL build translate rebuild 类似于编译（？）
KEIL output 需要输出isp可以编译的文件，需要调整这个

注意事项

while循环后面的括号不能为空，要让他循环可以在里面写上”1”

#include <intrins.h> //“nop” need this

对于发光二极管，1111 1110而不是0000 0001的时候，一个灯亮，只有当输入数字为0的时候，灯才会亮，你可以理解为只有为0的时候才有电位差，二极管才会发光
