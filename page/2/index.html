引言

在个人博客中，图片的存储和访问一直是一个需要解决的问题。外链图床（比如我之前使用的imgse.com路过图床，需要开会员，不然每日限额很严重）可能失效或者加载慢，有时还会遇到跨域问题，对于想要在博客里面嵌入大量照片是不可行的。所以，参考网络教程以及AI辅助，我使用 GitHub 仓库存储图片，配合 PicGo 管理上传，再通过 CDN 服务加速访问，搭建了一个长期可用、方便管理的个人博客图床。

护城河有什么用？

发表于 2026-03-22 更新于 2026-03-15 分类于 知识

概念

我们来看看wikipedia怎么说的吧：

护城河，亦作城壕或城濠，是古时由人工挖凿，环绕整座城、宫城、寺院等主要建筑的河，具有防御作用，可防止敌人或动物入侵。

世界各国在古代已有开凿护城河。在中国大陆北京的紫禁城、台湾新竹古城、左营旧城、亿载金城、日本的古城如松本城、江户（今东京）的皇居、以至欧洲各国的城堡及皇宫等等地方都建有护城河。欧洲不少城堡在护城河上建有可升起的木桥，以方便出入，亦可防止敌人进入。特别情况下，会选择湖泊作为城濠。如明初修建南京城墙时，放弃金川河，选择玄武湖。

当然，这只是实际意义上的护城河，还有抽象意义上的护城河，巴菲特用生动的比喻描述了这一概念：“奇妙的、由很深、很危险的护城河环绕的城堡。城堡的主人是一个诚实而高雅的人。城堡最主要的力量源泉是主人天才的大脑；护城河永久地充当着那些试图袭击城堡的敌人的障碍。”

护城河怎么发挥作用？

那么，护城河具体是如何发挥作用的？让我们通过两个真实的企业案例来理解:

可口可乐：护城河核心是无与伦比的品牌认知 + 轻资产生态模式。

消费者对口感形成心理忠诚，全球碳酸饮料市场份额46%，快餐与影院渠道覆盖率极高。公司只专注浓缩液生产与品牌，全球225家装瓶伙伴负责灌装分销，母公司掌控定价、包装与战术。这种模式带来60%+毛利率、连续60年股利增长。竞争者难以复制其全球品牌影响力和装瓶网络。

山姆会员商店：护城河建立在极致供应链精选 + 会员生态。

SKU仅3500-5000个，每类只选1-2最佳单品，通过买断独家、自有品牌Member’s Mark（占比30%+）、规模采购实现品质与低价。会员数从2018年300万增至2024年近900万，续卡率86%，年费收入超23亿。消费者为“确定性”（随便选也高性价比）买单。但近期选品泛化、质量问题动摇信任，山姆通过全球直采（如独家孔泰奶酪、树顶到卖场榴莲）、“先仓后店”前置仓+电商（占比超50%）持续加宽护城河，2025Q2净销售额仍增长30.1%。

共性逻辑：

  1. 创造定价权（高毛利或会员费）；
  2. 提高进入壁垒（难以快速复制）；
  3. 必须每年加宽（巴菲特强调：护城河不断拓宽，企业才能长期优秀）。

在快速变化的时代，个人也需护城河来获得安全感与主动权。它让你不可替代，专注创造而非焦虑竞争。

麦克利兰“冰山模型”揭示构成：

  • 上层（显性）：知识与技能（如编程、英语）。
  • 中层：能力（如沟通、结构化思维、情绪管理）。
  • 底层（最稳固）：价值观、性格特质、动机，决定你独特优势与能做成什么。

构建路径：从底层发掘优势 → 积累知识技能 → 形成难以撼动的核心竞争力。

所以我们应该：

  1. 发掘优势 —— 德鲁克“回馈分析法”：每次重要决定前记录预期结果，9-12个月后对比实际，持续执行即可发现真正长处（也可回顾过去成功案例）。

  2. 优势与社会需求匹配：单纯擅长不够，必须问：这个优势如何创造价值？如何满足市场/趋势需求？如何与个人目标统一？三者交汇处才是护城河落脚点。

  3. 纵横积累模型：

    • 横向：深耕当下价值创造（为客户/组织提供的核心输出），专注细节、持续1%优化。
    • 纵向：为横面服务而积累相关知识/能力（直接相关优先，逐步扩展能力圈）。所有学习都围绕“能为他人创造的价值”展开，避免盲目杂学。

  4. 持续加宽（像企业一样）：

    • 终身学习，避免知识过时（如柯达胶卷被颠覆）。
    • 构建组合优势：单一元素难胜，排列组合形成独特性。
    • 建立反馈闭环：定期检视、根据环境调整方向。

总结

护城河的终极意义在于创造的自由：企业无需时时防御，可专注为顾客创造更大价值；个人无需焦虑淘汰，可专注发挥独特优势。可口可乐与山姆的护城河皆经数十年沉淀，个人亦然。从今天开始，用回馈分析发掘长处，将优势对接社会需求，纵横积累、持续加宽——当你站起身回望，那条属于自己的宽阔护城河，将成为最值得骄傲的资产。

所以写一篇文章、读一篇文章不要只是感动自己，改变自己才是最真的，自己理解才是重点。

用我自己的话来说，护城河就是一个人（或一家企业）在这个世界里能够安全立足、别人暂时取代不了的那个“东西”。它从来不是单一的技能或天赋，而是一套多维度能力的组合拳——当几个看似普通的点叠加在一起，就形成了别人很难抄的独特壁垒。

拿企业举例，可口可乐的护城河不是单纯的饮料好喝，而是“品牌认知度极高 + 轻资产模式”的组合：它只掌控最核心的配方和品牌，把生产、灌装、分销这些中低端环节外包给全球225家区域伙伴，自己专注定价权和生态掌控。这套模式让竞争者望尘莫及，毛利率常年60%以上，连续60年涨股息。山姆会员店则是另一套组合：极致精选SKU（3500–5000个，每类只挑1–2个最好）+ 自有品牌+ 会员付费买“确定性”+ 全球直采冷链。这种“少而精 + 会员锁仓”的模式，也让模仿者短期内跟不上。

放到个人身上，我觉得道理是一样的。如果我只是一个会写代码的普通程序员，但社交能力弱、表达不行、动机也不强，那我的竞争力就很单薄，随时可能被更全面的人取代。但如果我把编程技能 + 良好的沟通表达 + 懂一点业务逻辑 + 强烈的成就动机 + 靠谱的性格叠加起来，这个组合就很难被快速复制——这就是我的护城河。

作为大三学生，我现在的护城河其实还很浅，甚至可以说还没真正挖起来。主要有两个明显短板：

  1. 知识和技能还停在表面：学过的东西不少，但都浅尝辄止，没有真正做成过有分量的项目，也没有形成体系。
  2. 动机非常薄弱：我还没搞清楚自己到底想成为什么样的人、想为什么而努力。每天上课、刷题、像在完成任务，而不是在为某个清晰的方向积累。

意识到这些后，我开始试着把“护城河”这个概念真正融入日常生活，而不是停留在想一想就结束的阶段。我给自己定了一些小而具体的行动：

  • 每天花10分钟写“动机日志”：今天什么事让我有点兴奋？什么事情让我很沮丧、无法集中注意力？为什么？慢慢找底层驱动力。
  • 每周挑一个技能做“项目驱动练习”：不是只看教程，而是逼自己输出一个小东西（比如建一个模型、设计一个产品），哪怕很粗糙。
  • 主动把技能“交叉”：比如写技术类博客，锻炼自己的表达能力的同时提升自己的技术素养、帮朋友解决实际问题，刻意练沟通等。
  • 记得复盘：列出当前的能力清单，问自己“这些点怎么组合才能让我更独特？”同时对比预期和实际结果，调整方向。

护城河不是一夜之间建成的。可口可乐用了上百年，山姆也迭代了近30年。我现在才大三，时间是我最大的优势。也许几年后回头看，现在这些小小的刻意练习，就是那条河的起点。

我希望未来的自己，能站在一条不算太宽但足够深的护城河边上，对着世界说： “来吧，我在这里，有我自己的位置。”

你要如何设计自己的人生，这件事完全取决于你自己。

复利究竟有什么魔力？

发表于 2026-03-22 更新于 2026-03-30 分类于 知识

复利是什么

我们来看看wikipedia怎么说的吧：

复利率法[1]（英文：compound interest），是一种计算利息的方法。按照这种方法，利息除了会根据本金计算外，新得到的利息同样可以生息，因此俗称“利滚利”、“驴打滚”或“利叠利”。只要计算利息的周期越密，财富增长越快，而随着年期越长，复利效应亦会越为明显。

说人话，我的理解就是：你投资产生的收益，你要忍住不要直接花掉，而是要连本带利继续投资，这样的话，即使你的收益率不高，但是财富依然会有比较可观的增值。

复利怎么运作

复利和单利的区别在于，收益的那一部分钱是可以进行再投资的

  • 单利：假设你每年投入10000元，年利率是10%，那么十年以后的收益是10X10000X10%=10000元
  • 复利：同样的条件下，你的十年收益是 10000X(1+10%)^10-10000=15937元

相比下来，整整多了5937元！

诶你有可能会说，这也太少了吧，十年，整整十年我才赚了5937元，累死累活，挣个毛线啊…

说得对，复利的魅力，在这个举的例子里面体现的还不够好。

你想想，如果你把里面的投入本金换成10000万（也就是1亿元），你想想你多赚了多少，不出意外的话，复利比单利多了接近六千万吧

我们不妨把视线放长一点，假设我们是一个很老练的投资人，能够保持年利率10%下去30年，我们再算一算这笔账：

  • 单利：30年收益是30000元
  • 复利：30年收益是10000X(1+10%)^30-10000=164494元

可以窥见的是，随着时间愈长，雪球滚得越大，这是美丽的“雪球效应”

以上还是基于年复利率的前提下，你想想，如果是月复利率、日负利率，是不是不用30年就能大赚一笔？

总结一下，影响复利效应的关键因素

  • 投资期限长短：资金滚得越多，复利效应就越明显
  • 收益率的高低：自然是越高越好
  • 本金大小：本金越多，自然赚的越多
  • 再投资频率：年复利、月复利、日负利，再投资频率越高，复利效益就越明显

天使和魔鬼

复利能让你变得很富有

以香港首富李嘉诚为例，1950年以7,000美元成立长江塑胶厂，在2006年拥有约188亿美元身家计算，撇开其他因素，他的财富在57年增长268.6万倍

也能让你被薅得很穷

高利贷就是很好的例子，现时世界各国普遍都有对放债人所收取的利息有最高限制，一般都以30%年息为上限，在此以上的都被视为进行高利贷活动而加以取缔。而现时信用卡欠账的年息往往高于20%甚至30%

延伸

对于投资的初学者来说，复利是一个很好的认识投资的工具，他让我们意识到投资世界的魔力，复利是核心的财富增值工具。

可以从以下几个方面去运用好这个工具：

  1. 拒绝借高利贷等一切涉及复利的吃人的东西

  2. 不要追求过高的投资回报率，往往稳定的投资回报率更加重要

  3. 复利最青睐的就是时间，年轻的时候投入的一笔钱，如果可以保证稳定可观的回报率，在时间的见证下也能积累一笔很可观的财富

  4. 买一些优质的投资项目，能够保证复利有长期的时间可以去发挥作用而不会突然崩塌

    这一点和《穷查理宝典》里的“我们偏向于把大量的钱投在我们不用在另作决策的地方”不谋而合

  5. 尤其注意的是，复利的基础是你的本金，所以，当你的收益已经到达一定数目的时候，我们就可以留出自己的本金，保证自己不至于在钱桌上全盘皆输

怎么去优雅的转pt种？

发表于 2026-03-22 更新于 2026-04-03 分类于 方法

规则

北洋园pt自查

电影

  • [ ] 我确认不是禁转的种子，可以转种

  • [ ] 我确定种子文件如果是蓝光原盘的话，一定是有中文字幕的版本

很简单，看种子简介里面subtitles有没有chinese就可以了，没有中文字幕的蓝光原盘一定要上传到北洋园pt，不然你差一点就要上传成功了

  • [ ] 我确定蓝光原盘/remux资源如果在同等分辨率下没有版本

比如如果已经有1080p remux资源，你就不要在上传其他站点搬来的1080p remux资源，但是如果没有2160p remux资源你是可以搬来的

  • [ ] 我确认remux作品视频码率不低，1080p不低于4000kb/s，2160p不低于8000kb/s

种子的简介里面是有的，如果不是remux作品要求就没有这么严格啦

  • [ ] 我确定我上传的文件不是同一系列但未确定完结的电影合集，以及以某导演、演员等打包的电影合集（按套装售卖的电影合集除外）

  • [ ] 我确定mediainfo是放在截图的前面，简介后面而不是在海报前面

  • [ ] 我确定站内没有更好的源

btschool/52pt

btschool的发种有点水，发种最大的特点是要有一个投票环节（elite user以下需要，elite user以上可以直接发种），我发了几个种子，似乎很难不被通过，发种的步骤大差不差，可惜的一点是自动发种体验不好，因为用户等级不够似乎无法直接跳转到发种界面，只能曲线救国，在其他站点的发布界面复制内容去填写候选环节的一些信息了

新发现的auto_feed PT一键转种脚本这个脚本可以直接给你跳转到btschool候选界面的，这个脚本还是可以的哈哈

52pt也是要通过候选的，但是等你等级升上去了就可以不用候选

1ptba/soulvoice

发种更是水，似乎不需要经过审核环节，个人觉得是因为站点的种子数比较少，鼓励大家积极上传种子的，自然对于资源的品质要求就没有那么高了

mt/ttg

这两个都是大站，估摸着发种不会那么简单

自动发种

pt站点的自动转种推荐使用EasyUpload PT一键转种油猴脚本，它能实现“从源站一键跳转到目标站，并自动填充表单”，极大加速发种流程

由于在探索转种的过程中发现mteam上面这个脚本居然不支持，所以找到了一个支持mteam转种的脚本auto_feed PT一键转种脚本，这个似乎也不错

安装步骤

  1. 安装用户脚本管理器：Chrome/Edge 用 Tampermonkey（篡改猴），Firefox 用 Tampermonkey 或 Violentmonkey
  2. 访问脚本页面EasyUpload PT一键转种油猴脚本：点击“安装此脚本” → 确认安装（这一步可能需要科学上网的）

使用流程

  1. 打开源站的 torrent 详情页，会出现一个这样的小白花在转，点击小白花打开设置面板

  2. 转种站点启用: 定义需要在页面显示的转种站点列表

    点击单个的站点链接是实现单个站点转种，记住，点一次就行了，点太多会出一大堆的

  3. 批量转种启用: 点击页面的一键群转按钮将同时打开选中的站点转发

  4. 站点搜索启用: 定义需要在页面显示的快速搜索列表，搜索似乎是通过imdb编号来实现的，我在想，如果有一些站点没有这些编号的咋办

遇到的问题

  1. 如果发现转过去的图片无法显示怎么办？

    可以通过手动上传图片实现，复制无法显示的图片->https://iili.io/ ->把复制的图片上传->得到链接->链接就是红框那一段->bbcode使用[img]…[img]代码嵌入即可

写在最后

  • 任何情况下都不要完全依赖脚本，脚本不可能做到对每个站点都完美适配，发布前需要自己认真检查是否有填写错误或者不符合站点规则的地方，始终遵守目标站的发种规范
  • 插件目前对音乐、MV、动漫的种子转载不支持(分类可能不会自动填写)，需要自己手动填一下
  • 由于TTG的图片加载策略，需要等页面完全加载完整后再点击转载到其他站，否则种子信息会获取不完整
  • 转的种子不要乱转，最好是free的，如果计算了你的下载量的，你很亏的

写在前面

iPhone一直还停留在ios 18.7，很久没有自动更新过系统。、

iPad已经更新到了ipad os26.3，前段时间iPhone和iPad使用接力功能发现很不好用，就想着是不是可能是系统跨度大的问题。

iPhone要不要更新到ios 26.3？一直很犹豫，所以就写了这篇文章。

一些对比

最大的特点就是liquid glass的ui设计，整体的ui都变得圆润化，ui变成毛玻璃格式。

在视频里发现：ios26.3的电池最大容量是86%，ios18.7的电池最大容量是100%，这有点吓人，更新系统难道会让我的电池最大容量下降很多吗？

其实换个角度想想，右边的这个手机，不正好是我现在自己手机的电池最大容量吗？他既然可以很好的运行ios26，那我也不可以吗？哈哈哈

从视频很明显还能看出，ios26系统会更加的流畅，退出软件不会有ios18的那种顿挫感，而且进入app，可以明显分辨出ios26会更快一点。

结论

对于我个人而言，还是决定更新ios26.3.1，原因如下：

  • 和iPad能够更好的兼容，接力功能可以更好的实现
