Aria2 进阶：如何安全地为 Aria2-Explorer 扩展增加「删除任务并清理本地文件」功能

发表于 2026-07-03 更新于 2026-07-20 分类于 教程

在大数据与 PT 下载的时代，Aria2 凭借其轻量级、无图形界面低资源消耗，以及强大的 RPC 远程调用能力，成为了无数 NAS 玩家、Home Lab 爱好者以及下载狂热者的“万金油”神器。

然而，凡是使用过 Aria2 的朋友，几乎都遇到过一个极其头疼的痛点：在前端网页或浏览器扩展（如 Aria2-Explorer、AriaNg）中点击“删除任务”，仅仅是将该任务从 Aria2 的下载队列中抹去。原本下载在本地磁盘的大文件、分块碎片，以及伴随的 .aria2 控制文件，依然纹丝不动地躺在硬盘里。 日积月累，磁盘空间悄无声息地被这堆“幽灵文件”吞噬殆尽。

虽然该需求在 Aria2-Explorer 的 GitHub Issue 列表中高悬已久（例如经典的 #52「求添加删除任务时同时删除对应文件功能」），但由于各平台权限限制，官方至今没有提供原生支持。

今天，我们将从 Full-Stack 安全开发与系统防提权 的视角出发，彻底打通浏览器扩展与本地操作系统的壁垒，实现一套极度安全、防止命令注入、界面不卡死的本地文件自动清理方案。

丢掉手机，好好吃饭！

发表于 2026-06-15 分类于 生活
尝试吃饭时彻底丢掉手机，发现吃饭时间从半小时缩短到十分钟，脑子也更清醒了

手机等电子产品非常喜欢多管闲事，什么活动他都会插一嘴，踢一脚
今天吃饭尝试彻底丢掉手机，发现一个很大的变化：就是吃饭的时间缩短了一大堆，平常刷电子产品吃顿饭要半个小时，丢掉手机吃顿饭，只要十分钟，很吓人。

除此之外，丢掉手机以后，我发现脑子似乎更加清醒，脑子里面似乎会有说不完的问题和想法等着去解答，有数不完的领域可以去探索，回想之前刷着手机吃着饭，似乎脑子根本不会想那么多，就连吃的饭是具体什么味道都不清楚。

但今天不同，细细品味了吃的排骨玉米饭，注意到了里面的枸杞和葱花，注意到了入嘴的生姜，还在心里感触：小时候那个不吃葱不吃香料的孩子，现在长大了，不知道为什么想到这个，心里还有点酸酸的。

丢掉电子产品，好好吃饭睡觉，说的轻巧，做起来好难的。

不带手机吃饭，你怎么付钱？
